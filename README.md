Welcome to the first release of Instaco.
Instaco is a comparator between Instagram followers and following without API uses, or third parties apps.

To understand how you can scrap your followers, and following lists, check the guide at the following link: https://www.francescoascenzi.com/blog/articles/index.php?p=general&t=what-is-instaco-and-how-to-use-it&i=2
